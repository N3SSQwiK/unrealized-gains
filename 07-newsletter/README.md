# [Newsletter Name] — AI Newsletter

Monthly internal AI newsletter.

## Pipeline

```
ideas/          Topic ideas captured as they come — one file per idea
drafts/         Articles in progress
editions/       Published editions (archive)
```

## Workflow

1. **Capture** — When you encounter something relevant (meeting, article, tool, announcement), drop it in `ideas/` or tag an INBOX.md entry with `newsletter`
2. **Curate** — Mid-month, review ideas and pick 3-5 for the edition
3. **Draft** — Write articles from selected ideas
4. **Assemble** — Compile into the edition template
5. **Publish** — Send it out, archive the edition
